# Node dependencies:
```
npm i express
npm i mysql
npm install cors
```
## FastAPI:
```
pip install fastapi
pip install mysql-connector-python
pip install hypercorn
pip install uvicorn
```
## Для запуска серверного приложения (без перехода в папку backend) для тестирования:
```
python -m uvicorn server:app --reload
```
